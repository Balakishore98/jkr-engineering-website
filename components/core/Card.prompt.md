White surface container — hairline border, crisp low shadow, low radius.

```jsx
<Card accent hover>
  <h3>5-Axis Milling</h3>
  <p>Complex geometries held to ±0.01&nbsp;mm.</p>
</Card>
```

- `hover`: lift + deepen shadow on hover (use for clickable capability cards)
- `accent`: amber rule along the top edge
- `padding`: any CSS padding value
